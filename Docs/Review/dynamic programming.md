# 动态规划
## 核心思想
- **要保证当前计算的问题的全部子问题都已经计算过了, 并且每一个子问题已经是最优解!**
- 回溯时, 只能倒推(自顶向下), 因为每一个底问题都会推出一个最终解, 而仅有我们求出来的最终解是最优的, 因此要倒推。
